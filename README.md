# ربات اخبار روزانه‌ی انیمه (تلگرام)

این ربات هر روز به‌صورت خودکار (با GitHub Actions) به سایت‌های خبری انیمه سر می‌زنه،
خبرهای جدید رو پیدا می‌کنه، به فارسی ترجمه می‌کنه و برات توی تلگرام می‌فرسته.

منابع فعلی: Anime News Network، Anime Corner، Crunchyroll News
(منبع فارسی معتبر و به‌روزی که مخصوص «خبر» انیمه باشه — نه دانلود — پیدا نکردم،
برای همین از منابع انگلیسی معتبر استفاده می‌کنیم و خودمون ترجمه می‌کنیم. اگه
سایت فارسی خاصی مدنظرت هست بگو تا اضافه‌ش کنم.)

---

## مرحله ۱: ساخت ربات تلگرام

1. توی تلگرام به [@BotFather](https://t.me/BotFather) پیام بده.
2. دستور `/newbot` رو بفرست و اسم و یوزرنیم دلخواه رو بده.
3. یه توکن بهت می‌ده، شبیه این: `123456789:ABCdefGhIJKlmNoPQRstuVWXyz`
   👉 این رو جایی کنار بگذار (اسمش `TELEGRAM_BOT_TOKEN`).

## مرحله ۲: گرفتن Chat ID

1. با ربات تازه‌ساخته‌ت یه پیام (مثلاً `/start`) بفرست.
2. توی مرورگر این آدرس رو باز کن (بجای `<TOKEN>` توکن خودت رو بگذار):
   ```
   https://api.telegram.org/bot<TOKEN>/getUpdates
   ```
3. توی خروجی JSON دنبال بخش `"chat":{"id": ...}` بگرد. همون عدد، Chat ID توئه.
   👉 این رو هم کنار بگذار (اسمش `TELEGRAM_CHAT_ID`).

> نکته: اگه می‌خوای ربات توی یه گروه پیام بفرسته، ربات رو به گروه اضافه کن و
> Chat ID گروه (که معمولاً با علامت منفی شروع می‌شه) رو استفاده کن.

## مرحله ۳: بالا بردن روی گیت‌هاب

1. یه ریپازیتوری جدید (می‌تونه Private باشه) توی گیت‌هاب بساز.
2. تمام فایل‌های این پوشه رو توش آپلود کن (یا با git push).
3. برو به تنظیمات ریپو: **Settings → Secrets and variables → Actions → New repository secret**
   و این دو مقدار رو اضافه کن:
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`

## مرحله ۴: تست

توی تب **Actions** ریپو، وارد workflow به اسم **Daily Anime News** بشو و دکمه
**Run workflow** رو بزن تا یه بار دستی اجرا بشه و ببینی پیام‌ها درست میان.
بعد از این، خودش هر روز ساعت ۸ صبح (به وقت تهران) بدون هیچ کاری از طرف تو اجرا می‌شه.

---

## تغییر دادن ساعت اجرا

توی فایل `.github/workflows/daily-news.yml` این خط رو ببین:
```yaml
- cron: "30 4 * * *"
```
این یعنی ۰۴:۳۰ به وقت UTC (= ۸:۰۰ صبح تهران). اگه می‌خوای ساعت دیگه‌ای باشه،
عدد رو با کمک سایت [crontab.guru](https://crontab.guru) تغییر بده (به وقت UTC).

## اجرای روی کامپیوتر خودت (اختیاری)

اگه ترجیح می‌دی به‌جای گیت‌هاب، روی سرور/کامپیوتر خودت اجرا بشه:

```bash
pip install -r requirements.txt
export TELEGRAM_BOT_TOKEN="توکن‌ات"
export TELEGRAM_CHAT_ID="چت آیدی‌ات"
python bot.py
```

و با `cron` (لینوکس) یا Task Scheduler (ویندوز) هر روز اجراش کن.

## فایل‌های پروژه

- `bot.py` — کد اصلی ربات
- `requirements.txt` — کتابخانه‌های لازم
- `sent_ids.json` — لیست خبرهایی که قبلاً فرستاده شدن (برای جلوگیری از تکرار؛ خودکار آپدیت می‌شه)
- `.github/workflows/daily-news.yml` — تنظیمات اجرای خودکار روزانه
